# 001_Triangle
Renders a coloured triangle.

Uses the following third-party libraries:
- GLEW (http://glew.sourceforge.net/)
- GLFW (https://www.glfw.org/)
- GLM (https://github.com/g-truc/glm)

Also, uses a home-made library OglLib.
